seed 🌱

A vue-cli template who plants the first seed of your project in order to become a sturdy and flowering tree

Introduction

This vue-cli template is targeted towards single page applications with state management and support for Electron. In order to fully enjoy working with this template, please read these documentations :

  • ES2015 syntax
  • vue
  • vue-router
  • vuex
  • vue-i18n
  • electron
  • standard style
  • sass
  • successfull git branching model

How to use

Install vue-cli as global

$ npm install -g vue-cli

Init your project

$ vue init pierrechls/seed my-project     # init template
$ cd my-project                           # go to your project folder
$ npm install                             # install dependencies
$ npm run dev                             # serve with hot reload
